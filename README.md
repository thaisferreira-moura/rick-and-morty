## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
Este projeto foi desenvolvido com o objetivo de praticar o consumo de APIs REST utilizando JavaScript. A aplicação realiza requisições à API pública do Rick and Morty, exibindo informações dos personagens de forma dinâmica na interface.

Durante o desenvolvimento, foram aplicados conceitos como fetch API, manipulação do DOM, tratamento de respostas em formato JSON, além da utilização de HTML, CSS e JavaScript para construção da interface.
